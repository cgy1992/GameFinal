Write 3D Game in an Easier Way

View on GitHub Download Now View Tutorials

  • For Many Kinds of Games

    GameFinal is an open-source but very powerful game engine. It is not limited to a specific kind of game development. You can make whatever you like. From exciting PFS game to fast racing game, from classical RPG to small indie game.

  • Easy Shader Programming

    Although GameFinal provides lots of useful built-in shaders and materials, it also gives freedom to write your own shaders. GameFinal allows you to get rid of the complicated graphics programming and use more friendly interfaces.

  • Character Animation

    GameFinal supports many different formats of 3D models. You can convert them to a special format which has faster loading and rendering performance. GameFinal also supports skeleton animations and helps you manage the complex animations.

  • Real-time Lighting

    GameFinal not only has realistic lighting effects, but also allows you to add many real-time light sources into the scene. Even though you add thousands of dynamic lights, GameFinal will ensure the game running in a high performance.

  • Outdoor Rendering

    Although outdoor rendering is a performance killer in many games, GameFinal can render magnificent outdoor scenery in a high performance no matter how large-scale your scene is. You just put all your energies in the design of the game world.

  • Geometry Complexity

    The engine uses efficient algorithm to do the scene management and has many strategies to accelerate the mesh rendering. You're allowed to add hundreds and even thousands of objects into the scene without noticing the performance loss.

Engine Publishing and Presentation

May 18, 2015

The GameFinal, the new open-source 3D game engine, was released to all the developers in May 18, 2015. An publishing presentation was held in the east campus of Sun Yat-sen University(SYSU), Guangzhou, China. Many students, software developers, as well as professors in CMU and SYSU attended the conference. The engine's author, Edison Gao, gave a brief introduction to GameFinal's architectures. Wenjun Li, the president of SYSU Software School, spoke highly of GameFinal: "This is a pretty interesting project and reflects many next-gen features" Wenjun Li said. Download PPT of the presentation.

Scene Editor and Physics Editor's Trial Version Published

GameFinal is trying to provide game developers with powerful and complete tool chains. For this purpose, a scene editor was released. Developers and designers are able to design the scene and level of the game completely in the scene editor even without writing any code; thus they can put more energies on the game design rather than a large number of programming interfaces. A physics editor is also provided for developers to bind physics to objects. Because these two editors are still in beta test, they are not included in the download zip. Users can download a trial version from github.
