Contribute to GameFinal:

Everyone is welcome to contribute to the GameFinal Engine. This project is open source, and it is made better in every possible way thanks to contributors like you.

If you meet one or more of the requirements, welcome to contact with the author and join the GameFinal project:

  • An avid gamer with a passion for making games.
  • Have deep understanding in C++, data structures and algorithms
  • Familiar with computer graphics
  • Know the basic knowledge of Shader programming and Cg/HLSL languages
  • Ability to do performance analysis and optimization of game and graphics architecture.

Your contributions might include (but not limited to) core libraries' development, tools development, game resources, demos, documents and bug reporting.
